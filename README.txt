Porfessional Blog Post Layouts For WordPress

== Description ==

Postify is a professional post layout plugin. The plugin will allow you to create differen types of posts layouts in your WordPress site with very simple way. You will able to create grid and carousel showcase layout for your blog posts. You just need to use simple shortcode in your page or post to show your posts.

Plugin features:

*   3 Grid layout with different columns
*   3 carousel layout
*   Use as post per page
*   Can be set excerpts length
*   Navigation/pagination show and hide option for carousel layout
*   Loop option for carousel layout

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `postify.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use [postify layout="grid"] shortcode to show grid layout.
